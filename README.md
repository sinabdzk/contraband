# Contraband

Projet web front-end.

## Prerequis

- Node.js 18+ (ou version recente)
- npm

## Installation

```bash
npm install
```

## Lancer le projet

```bash
npm run dev
```

## Structure

- `index.html`: point d'entree de l'application
- `css/style.css`: styles
- `js/app.js`: initialisation
- `js/data.js`: donnees
- `js/engine.js`: logique
- `js/state.js`: gestion d'etat
- `js/ui.js`: interface utilisateur
